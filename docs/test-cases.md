# 问卷系统测试用例

## 1. 测试说明

本文档用于验证基于 MongoDB 的问卷系统第一阶段功能是否满足要求。
测试方式包括接口测试和数据库结果验证。
接口测试工具使用 Thunder Client。
数据库验证工具使用 MongoDB Compass。

## 2. 测试环境

- 操作系统：Windows 10
- 开发工具：VS Code
- 后端：Node.js + Express
- 数据库：MongoDB
- 测试工具：Thunder Client
- 数据库可视化工具：MongoDB Compass
